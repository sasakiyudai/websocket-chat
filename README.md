#### setting
add .env in root like below for OAuth
```
FACEBOOK_API_CLIENT_ID=""
FACEBOOK_API_SECRET_KEY=""
GOOGLE_API_CLIENT_ID=""
GOOGLE_API_SECRET_KEY=""
GITHUB_API_CLIENT_ID=""
GITHUB_API_SECRET_KEY=""
```
